Support

Need help with FitFrame? Start with the common topics below or contact support directly.

I cannot import a photo

Check that Photos permission is enabled for FitFrame in iPhone Settings. Then reopen the app and try selecting the image again.

Export failed

If export fails, try again with a lower quality setting, then save once more. Also confirm that Photo Library access is allowed for saving.

Which preset should I use?

Use the social preset that matches where you plan to post. FitFrame includes popular ratios like 4:5, 1:1, 1.91:1, and 9:16.

Can I keep the whole photo visible?

Yes. FitFrame is built around a fit-first workflow, so the full image is preserved by default before you make zoom or position adjustments.

How do custom ratios work?

Open the custom ratio option, enter width and height values, then apply the new ratio to the current image.
